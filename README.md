# EDTF

An Extended Date Time Format (EDTF) / ISO 8601-2 parser and English language rendering
toolkit for Elixir.

## Compatibility

### EDTF / ISO 8601-2
EDTF.js fully implements [EDTF](http://www.loc.gov/standards/datetime)
levels 0, 1, and 2 as specified by ISO 8601-2
